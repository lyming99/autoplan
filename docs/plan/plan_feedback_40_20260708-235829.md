# 修复 MCP 配置显示密钥无内容

## 需求概要

反馈指出在 MCP 配置中点击“显示密钥”后没有显示已保存密钥。现有前端表单出于安全设计不从快照回填明文 authToken，眼睛按钮仅切换当前输入框类型，导致已保存密钥场景下仍为空。计划补齐受控的密钥读取通道，并调整 MCP 设置面板的显示逻辑，让用户能在明确点击显示后看到已保存密钥，同时保持默认脱敏、不误触保存覆盖。

## 任务拆解

  - 验收要点：未设置密钥时接口返回空值或明确的未设置状态，不抛出异常影响设置页
  - 验收要点：接口不改变现有 mcp:status 和 snapshot 的默认脱敏契约，普通快照仍不包含明文密钥
