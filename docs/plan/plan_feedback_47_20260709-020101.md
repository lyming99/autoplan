# 隐藏工作区终端和执行器功能入口

## 需求概要

根据反馈 #47，需要在工作区前端隐藏“终端”和“执行器”功能。结合当前代码，终端和执行器主要通过侧边栏导航、URL tab 深链和 WorkspacePage 条件渲染暴露；本计划聚焦隐藏用户可见入口并阻断直接 tab 访问，保留既有后端服务和数据结构，降低对自动任务、脚本、历史数据的影响。

## 任务拆解
