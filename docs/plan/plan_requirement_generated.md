## 需求分析与方案

### 需求概述

用户需要一个「需求整理与开发计划生成」功能模块，能够自动接收需求/反馈输入，分析其内容，并生成结构化的开发计划（包含任务拆解、验收要点、scope 标注等），最终将计划写入指定文件。

### 关键能力

1. **需求/反馈接收**：支持通过消息传递接收需求正文及关联附件。
2. **需求分析**：理解需求内容，判断影响范围。
3. **计划生成**：按 AutoPlan 标准格式输出 `## 任务拆解` 章节及合规任务行。
4. **文件写入**：将生成的计划写入到指定的输出文件路径。
5. **格式校验**：校验生成计划是否包含必需章节和合规任务行。
