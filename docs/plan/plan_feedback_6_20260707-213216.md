# 阻止 validation_failed 计划无限重跑占用项目队列

## 需求概要

反馈指出长耗时并非当前计划任务执行慢，而是同项目队列中前一个计划在完整验收失败后仍被调度器反复选中，导致 validation_failed 状态持续重跑并阻塞后续计划。需要把完整验收失败后的计划从自动调度队列中摘除，保留失败状态与日志供人工处理，并提供明确的重试/恢复入口，避免单个失败验收无限占住同项目串行执行队列。
