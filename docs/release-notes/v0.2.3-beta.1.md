# AutoPlan v0.2.3-beta.1

> **Beta 未测试版本**：这是面向快速分发的预发布版本，未经过完整人工测试，不等同于稳定正式版，也不承诺零缺陷。

## 近期更新摘要

- 整合远端已合并的 Linux 构建类型修复，包含 `.nvmrc`、lockfile 与 `aiConfigUpdate` 相关类型适配。
- AI 配置改为全局配置池，支持多配置、保留空 API Key 的编辑语义，并修复创建/更新失败时的错误提示与状态刷新。
- 对话模块增强项目隔离、多会话绑定、自动标题、置顶排序、当前 AI 配置切换与思考深度更新。
- 对话界面按最新设计稿重做，消息气泡、工具调用详情、侧栏对话高亮、整行点击和设置页编辑弹窗都做了交互整理。
- 工作区搜索弹层改为全局 Portal 定位，左侧面板支持独立滚动与尺寸调整，项目路径入口支持链接式打开。
- 补充多组回归测试与 smoke 覆盖，并同步计划文档记录近期需求/反馈处理过程。

## Beta 风险提示

本 beta 版本用于更快分发近期修复和功能更新。当前策略是：核心检查、构建和 smoke 通过后即可发布，以减少等待时间。因此，下载测试或日常使用中仍可能遇到功能缺陷、平台差异或安装包问题。
