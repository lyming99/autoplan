# AutoPlan v0.2.3-beta.7

> **Beta 未测试版本**：这是面向快速分发的预发布版本，未经过完整人工测试，不等同于稳定正式版，也不承诺零缺陷。

## 近期更新摘要

- 增强计划生成与执行链路：结构化计划规约解析、计划生命周期管理、任务执行会话上下文与快照处理进一步补全。
- 扩展任务 Scope 文件查看能力：任务列表支持在工作区内直接打开关联文件，可通过系统默认、文件夹定位、VSCode 或自定义命令等多种方式打开。
- 改进工作区界面：计划列表、Composer、聊天视图、项目页面和终端会话等界面与交互优化。
- 补充 AI 配置（ChatProvider 类型）、MCP 工具文档和接纳（acceptance）流程相关处理与测试覆盖。
- 更新开发脚本、冒烟测试与架构文档，新增近期计划与反馈记录。

## Beta 风险提示

本 beta 版本用于更快分发近期修复和功能更新。当前策略是：AI 修复问题后，只要项目编译通过即可发布，以减少人工测试和等待时间。因此，下载测试或日常使用中仍可能遇到功能缺陷、平台差异或安装包问题。

## 问题反馈

如果你发现问题，请直接到 GitHub Issue 界面提交反馈，并尽量包含使用的操作系统、安装包类型、复现步骤、截图或日志。后续 AutoPlan/AI 流程会持续从 Issue 中发现问题，生成修复计划并继续处理。
