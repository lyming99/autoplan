# AutoPlan v0.2.1-beta.1

> **Beta 未测试版本**：这是面向快速分发的预发布版本，未经过完整人工测试，不等同于稳定正式版，也不承诺零缺陷。

## 近期更新摘要

- 增强需求与反馈附件能力，支持文件选择、拖拽和截图粘贴，并把附件清单注入给 AI 工具作为上下文。
- 支持项目级与单条需求/反馈级 CLI 后端配置，可在 Codex CLI 与 Claude CLI 之间切换，并保留计划级 CLI 快照。
- 新增本机 MCP 外部接入能力，方便其它 MCP 客户端创建项目、提交需求和提交反馈。
- 完善计划任务 scope 规范、并发建议规则和 scope 文件打开配置，降低任务并发执行时的冲突风险。
- 补充三端发布流程与 macOS 发布安装说明，便于通过 GitHub Actions 生成 Windows、macOS、Linux 安装包。

## Beta 风险提示

本 beta 版本用于更快分发近期修复和功能更新。当前策略是：AI 修复问题后，只要项目编译通过即可发布，以减少人工测试和等待时间。因此，下载测试或日常使用中仍可能遇到功能缺陷、平台差异或安装包问题。

## 问题反馈

如果你发现问题，请直接到 GitHub Issue 界面提交反馈，并尽量包含：

- 使用的操作系统与安装包类型。
- 复现步骤、预期行为和实际行为。
- 相关截图、日志或附件。

后续 AutoPlan/AI 流程会持续从 Issue 中发现问题，生成修复计划并继续处理。
